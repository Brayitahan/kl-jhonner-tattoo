# kl_jhonner Tattoo & Piercing

**Problema:** El artista Jhonner Vargas carecía de presencia digital para exhibir su portafolio de tatuajes y servicios de piercing, dependiendo únicamente de recomendación boca a boca.

**Solución:** Landing page moderna y responsiva con galería visual de trabajos, sección de servicios detallados, información del artista y formulario de contacto. Diseño oscuro temático con tipografía personalizada (Oswald, Work Sans, Miltonian Tattoo) y experiencia de carga optimizada con skeleton screen.

**Stack:** HTML5, CSS3, JavaScript vanilla, diseño responsivo.
